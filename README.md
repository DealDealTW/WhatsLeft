# WhatsLeft

WhatsLeft 是一個幫助用戶追蹤食品和家居用品到期日的應用程序，旨在減少浪費並提高家庭庫存管理效率。

## 功能特點

- **庫存管理**：添加、編輯和刪除食品和家居用品
- **到期日追蹤**：設置和監控物品的到期日
- **購物清單**：創建和管理購物清單
- **多語言支持**：支持英文和中文（繁體和簡體）
- **家庭尺寸調整**：根據家庭人數自動調整物品數量
- **語音輸入**：通過語音快速添加多個項目
- **條形碼掃描**：掃描條形碼快速識別產品

## 技術堆棧

- React
- TypeScript
- Tailwind CSS
- Supabase (後端)
- Lucide Icons
- PWA 支持

## 安裝和運行

```bash
# 安裝依賴
npm install

# 運行開發服務器
npm run dev

# 構建生產版本
npm run build
```
